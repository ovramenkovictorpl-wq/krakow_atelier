# Atelier Kraków - Веб-сайт ательє

## Опис проекту
Багатомовний веб-сайт для ательє з ремонту та реставрації одягу в Кракові. Статичний сайт з підтримкою трьох мов (українська, польська, англійська).

## Технології
- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+)
- **Стилізація**: Custom CSS з CSS-змінними
- **Шрифти**: Google Fonts (Playfair Display, Inter)
- **Іконки**: SVG inline
- **Інтернаціоналізація**: Власна система перекладів через JavaScript

## Структура файлів
```
studio in Krakow/
├── index-new.html          # Головна сторінка
├── prices-new.html         # Сторінка з цінами на послуги
├── portfolio-new.html      # Портфоліо робіт (до/після)
├── contacts-new.html       # Контактна інформація
├── styles-new.css          # Глобальні стилі
├── script-new.js           # JavaScript логіка та переклади
└── images/                 # Зображення та логотипи
    └── logo_without_fon.png
```

## Основні функції

### 1. Багатомовність
- Підтримка UA/PL/EN через перемикач у хедері
- Переклади зберігаються в `script-new.js` у об'єкті `translations`
- Динамічна зміна контенту через атрибути `data-i18n`

### 2. Навігація
- Фіксований хедер з прозорістю та blur-ефектом
- Burger-меню для мобільних пристроїв
- Активний стан для поточної сторінки

### 3. Дизайн
- **Кольорова схема**:
  - Primary Dark: `#1A1A1A`
  - Secondary BG: `#FAF9F6` (кремовий)
  - Accent Gold: `#D4AF37`
  - Text Dark: `#2C2C2C`
  - Text Light: `#666666`
- Елегантний, мінімалістичний стиль
- Плавні анімації та переходи
- Responsive дизайн

### 4. Сторінки

#### Головна (index-new.html)
- Hero-секція з фоновим зображенням
- Розділ "Про нас"
- Сітка популярних послуг (6 карток)

#### Ціни (prices-new.html)
- 6 категорій послуг:
  - Джинси та штани
  - Сукні, спідниці та сорочки
  - Піджаки та трикотаж
  - Верхній одяг
  - Шкіра та хутро
  - Інше
- Стартові ціни з дискламером

#### Портфоліо (portfolio-new.html)
- Галерея робіт "до/після"
- Фільтрація за категоріями
- Модальне вікно для перегляду зображень

#### Контакти (contacts-new.html)
- Контактна інформація
- Соціальні мережі
- Можливо форма зворотного зв'язку

### 5. Соціальні мережі
- Instagram
- Telegram
- WhatsApp
- Іконки в хедері та футері

## Запуск проекту
Статичний сайт, відкривається безпосередньо через браузер:
```bash
# Відкрити index-new.html у браузері
start index-new.html
```

Або через локальний сервер:
```bash
# Python
python -m http.server 8000

# Node.js (npx)
npx serve
```
